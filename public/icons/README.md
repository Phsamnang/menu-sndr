# Icons Directory

This directory contains SVG icons used in the application.

## Online Presence Icon

The `online-presence.svg` icon is used in the menu page header.

### To Replace with Freepik Icon:

1. Go to: https://www.freepik.com/icon/online-presence_12914661
2. Download the icon as SVG format
3. Replace the file `public/icons/online-presence.svg` with the downloaded file
4. Make sure the SVG has proper viewBox and dimensions
5. The icon will automatically appear in the header

### Current Usage

The icon is displayed in the Header component (`app/components/Header.tsx`) next to the menu title.
